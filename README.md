# characterRogue

CUIで遊べる、1人用ローグライクボードゲームです。  
ランダム生成されるフロアを探索し、敵やボスを撃破しながらキャラクターを強化して深層を目指します。

## 実行方法

### 必要環境
- Python 3.10+

### 起動

```bash
python game.py
```

### テスト

```bash
python3 -m unittest discover -s tests -v
```

---

## タイトル画面

起動後はタイトル画面が表示され、次の項目を選べます。

- **Start**: ゲーム開始
- **Change Difficulty**: 難易度変更（Easy / Normal / Hard / Lunatic）
- **Quit**: 終了

---

## 基本操作

- `w` / `a` / `s` / `d`: 移動（敵にぶつかると通常攻撃）
- `.`: 待機
- `t`: Arcana（魔法）を使用
- `i`: インベントリ表示
- `u`: アイテム使用
- `k`: スキルツリーを開く（SP消費）
- `l`: ターンログ表示
- `p`: 詳細ステータス表示
- `h`: ヘルプ表示

---

## 画面表示とアイコン

- `#`: 壁
- `.`: 床
- `,`: ボスレーザー予告ライン
- `@`: プレイヤー
- `E`: 敵
- `M`: 中ボス
- `B`: ボス
- `I`: アイテム
- `F`: 友好キャラクター
- `>`: 階段

ステータスは3行表示です。

1. Floor / Moves
2. HP / MP / SP
3. Lv / XP / ATK / DEF / 装備アクセサリ

## 進行ルール

### 戦闘とダメージ
- 通常ダメージ式: `max(1, attacker_atk - defender_def)`
- HP が 0 以下になると撃破
- プレイヤー HP が 0 以下でゲームオーバー

### 階層進行
- 階段 `>` で次フロアへ進行
- **5, 15, 25...階**: 中ボス階（撃破まで階段封印）
- **10, 20, 30...階**: ボス階（撃破まで階段封印）
- **11, 21, 31...階**: 新しい周回
  - マップ横幅が拡張
  - 敵ステータスが強化

### スコア
到達階層・レベル・最終ステータス・所持Arcanaレアリティなどで計算されます。

---

## 成長要素

### レベルアップ
- 敵撃破・フロア殲滅ボーナスでXP獲得
- レベルアップ時:
  - 最大HP/最大MP/ATK/DEFが成長
  - HP/MP全回復
  - SPを1獲得

### スキルツリー（`k`）
- `v` Vitality: 最大HP +3
- `s` Strength: ATK +2
- `g` Guard: DEF +2
- `a` Arcane: ランダムArcanaを習得（1枠のため入れ替えあり）

---

## アイテム

### 消費アイテム
- `Potion`: HP回復
- `Power`: ATK上昇
- `Shield`: DEF上昇
- `Ether`: MP回復
- `Throwing axe`: 指定方向の直線上、最初の敵へ割合ダメージ
- `Bomb`: その階の全敵に割合ダメージ

### レアリティ
- Common / Uncommon / Rare / Epic / Legendary
- 高レアほど出現率が低く、性能が高い

---

## アクセサリ

- `Lucky amulet`: 獲得XP倍率アップ
- `Kote`: ATK/DEF上昇
- `Vampire's Fang`: HP回復の超過分をMPへ変換
- `Dark Wizard's Staff`: MP回復の超過分をHPへ変換
- `Guardian's Armor`: 待機でDEF倍率スタック、移動で倍率リセット
- `Berserker's club`: 最大MPを0にし、ATK倍率アップ
- `Roller shoes`: 1ターンの連続移動可能入力数を増加
- `Gunpowder box`: **通常攻撃またはArcanaで敵を倒した際、超過ダメージを周囲1マスへ連鎖爆発として伝播**

---

## Arcana（魔法）

Arcanaは **1つだけ所持** できます（新規習得時に入れ替え確認）。

- `Comet Missile`: 直線上の敵1体に攻撃
- `Flare Curtain`: 周囲1マスの敵を同時攻撃
- `God's Wrath`: 1ターン詠唱後に高威力攻撃
- `Healing`: 自身のHP回復
- `Vampire Kiss`: 隣接敵に攻撃＋吸収回復
- `Frugal soul`: 次に使うアイテムの消費を確率で無効化

---

## 友好キャラクター（`F`）

各フロアに以下3人が出現（各1回まで）。

- **merchant**
  - アイテム1つを、ランダムな別アイテムに交換
- **technician**
  - アイテム1つを、ランダムなArcanaに変換
- **maze traveler**
  - 次に階段を使ったときの移動先を「現在階層の2倍」または「半分」に変更

---

## ボス行動

ボスは状況に応じて以下を使用します。

- 近接攻撃
- エリート敵召喚
- レーザー予告 → 次ターン照射
- HP低下時の1回限定強化（回復 + 能力上昇）
