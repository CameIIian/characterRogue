# characterRogue

CUIで遊べる、1人用ローグライクボードゲームです。  
ランダム生成されるフロアを探索し、敵・中ボス・ボスを撃破しながら、できるだけ深い階層を目指します。

## 実行方法

### 必要環境
- Python 3.10+

### 起動

```bash
python game.py
```

### テスト

```bash
python3 -m unittest discover -s tests -v
```

---

## ゲームの流れ

1. タイトル画面で難易度を選択して開始
2. フロアを探索し、敵を倒しながらアイテムを収集
3. 階段 `>` で次フロアへ移動
4. 5階ごとに中ボス、10階ごとにボスと戦闘
5. レベルアップやスキル強化、Arcana習得でビルドを伸ばす

---

## 操作方法

- `w` / `a` / `s` / `d`: 移動（敵にぶつかると通常攻撃）
- `.`: 待機
- `t`: Arcana（魔法）を使用
- `i`: インベントリ表示
- `u`: アイテム使用
- `k`: スキルツリーを開く（SP消費）
- `l`: ターンログ表示
- `p`: 詳細ステータス表示
- `h`: ヘルプ表示

## 画面表示・UIの見方

### マップ記号
- `#`: 壁
- `.`: 床
- `,`: ボスレーザー予告
- `@`: プレイヤー
- `E`: 敵
- `M`: 中ボス
- `B`: ボス
- `I`: アイテム
- `F`: 友好キャラクター
- `>`: 階段

### ステータス表示（3行）
1. Floor / Moves
2. HP / MP / SP
3. Lv / XP / ATK / DEF / 装備アクセサリ

## 戦闘と進行ルール

### ダメージ計算
- 通常ダメージ: `max(1, attacker_atk - defender_def)`

### 撃破条件
- 敵HPが0以下で撃破
- プレイヤーHPが0以下でゲームオーバー

### 階層ルール
- 階段 `>` で次フロアへ進行
- **5, 15, 25...階**: 中ボス階（撃破まで階段封印）
- **10, 20, 30...階**: ボス階（撃破まで階段封印）
- **11, 21, 31...階**: 新しい周回
  - マップ横幅が拡張
  - 敵ステータス強化

## 成長システム

### レベルアップ
- 敵撃破・フロア殲滅でXP獲得
- レベルアップ時:
  - 最大HP / 最大MP / ATK / DEF が成長
  - HP / MP 全回復
  - SP +1

### スキルツリー（`k`）
- `v` Vitality: 最大HP +3
- `s` Strength: ATK +2
- `g` Guard: DEF +2
- `a` Arcane: ランダムArcanaを習得（1枠のため入れ替えあり）

---

## アイテムとアクセサリ

### 消費アイテム
- `Potion`: HP回復
- `Power`: ATK上昇
- `Shield`: DEF上昇
- `Ether`: MP回復
- `Throwing axe`: 指定方向の直線上、最初の敵へ割合ダメージ
- `Bomb`: その階の全敵に割合ダメージ

### レアリティ
- Common / Uncommon / Rare / Epic / Legendary
- 高レアほど出現率低・性能高

### アクセサリ
- `Lucky amulet`: 獲得XP倍率アップ
- `Kote`: ATK/DEF上昇
- `Vampire's Fang`: HP回復超過分をMPへ変換
- `Dark Wizard's Staff`: MP回復超過分をHPへ変換
- `Guardian's Armor`: 待機でDEF倍率スタック、移動でリセット
- `Berserker's club`: 最大MPを0にしてATK倍率アップ
- `Roller shoes`: 1ターン中の連続移動入力数を増加
- `Gunpowder box`: 撃破時の超過ダメージを周囲へ連鎖

---

## Arcana（魔法）

Arcanaは **1つだけ所持** できます（新規習得時に入れ替え選択）。

- `Comet Missile`: 直線上の敵1体に攻撃
- `Flare Curtain`: 周囲1マスの敵を同時攻撃
- `God's Wrath`: 1ターン詠唱後に高威力攻撃
- `Healing`: 自身のHP回復
- `Vampire Kiss`: 隣接敵に攻撃＋吸収回復
- `Frugal soul`: 次に使うアイテムの消費を確率で無効化

## 友好キャラクター

友好キャラクターはマップ上で `F` として出現し、接触すると取引やイベントが発生します。  
**1フロアあたり最大1人**、かつ**10階層（1サイクル）中に各役職は最大1回だけ**出現します。

### 出現率
- **merchant**: 10%
- **technician**: 10%
- **maze traveler**: 2%
- **Friendly demon**: 2%

（合計24%のため、何も出現しないフロアもあります）

### 役職ごとの効果
- **merchant**
  - アイテム1つをランダムな別アイテムに交換
- **technician**
  - アイテム1つをランダムなArcanaに変換
- **maze traveler**
  - 次の階段移動先を「現在階層の2倍」または「半分」に変更
- **Friendly demon**
  - 現在HPから任意量（ただしHP1は残す）を差し出して契約
  - 差し出した値を **ATK と DEF に自由配分**して上昇

## スコア

到達階層・レベル・最終ステータス・所持Arcanaレアリティ等をもとに算出されます。

深く潜り、効率よくビルドを伸ばすほど高スコアを狙えます。
