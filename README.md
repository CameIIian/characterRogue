# characterRogue

CUIで遊べるシンプルな1人用ローグライクボードゲームです。

## 実行方法

```bash
python game.py
```

## 操作

- `w` / `a` / `s` / `d`: 移動, 攻撃
- `f`: 隣接する敵へ攻撃（上下左右の順で最初に見つかった敵を攻撃）
- `t`: 魔法（テクニック）を使用
- `.`: 待機
- `i`: インベントリ表示
- `u`: アイテム使用
- `k`: スキルツリーを開いてSPを使用（`v`/`s`/`g`/`a`）
- `h`: 操作/アイコンのヘルプ表示
- `q`: ゲーム終了

## アイコン一覧

- `#`: 壁（通行不可）
- `.`: 床（通行可）
- `,`: ボスレーザー予告ライン（次のターンに照射）
- `@`: プレイヤー
- `E`: 敵
- `M`: 中ボス
- `B`: ボス
- `I`: アイテム
- `>`: 階段（次のフロアへ移動）

## ルール概要

- 初期ステータス: HP 10 / ATK 3 / DEF 1 / MP 5
- 敵に接触すると戦闘
- ダメージ式: `max(1, attacker_atk - defender_def)`
- 階段 `>` に到達すると次フロアへ進行（ゲームは終了せず、11階以降も継続）
- HP 0で敗北
- 中ボス階層（5, 15, 25, ...）は中ボス撃破まで階段が封印される
- ボス階層（10, 20, 30, ...）はボス撃破まで階段が封印される
- 10階層ごとの新しい周回開始時（11, 21, 31, ...）に以下を強化
  - 周回補正で敵ステータスを強化
  - マップ横幅を拡張

## アイテムの種類とレアリティ

- アイテム種類: `Potion` / `Power` / `Shield` / `Ether`
  - `Potion`: HP回復
  - `Power`: ATK上昇
  - `Shield`: DEF上昇
  - `Ether`: MP回復
- レアリティ: `Common` / `Uncommon` / `Rare` / `Epic` / `Legendary`
- レアリティが高いほど出現率は低く、効果量は高くなります。

## レベルアップとスキルツリー

- 敵撃破でXPを獲得し、しきい値到達でレベルアップ
- レベルアップ時にHP/MP/ATK/DEFが成長し、HP/MPは全回復
- レベルアップごとにSPを1獲得し、`k`でスキル強化
  - `v`: Vitality（最大HP増加）
  - `s`: Strength（ATK増加）
  - `g`: Guard（DEF増加）
  - `a`: Arcane（魔法を習得）

## 魔法（テクニック）

- `Comet Missile`: 直線上の最初の敵を攻撃
- `Flare Curtain`: 自分の周囲1マス以内の敵を同時攻撃
- `God's Wrath`: 1ターン詠唱後に高威力攻撃
- 魔法はMPを消費し、レアリティで威力/消費が変化

## ボス行動について

- ボスは近接攻撃・召喚・レーザー予告を使用
- レーザー予告ターンはライン上に `,` が表示され、次のターンに照射されます
