# characterRogue

CUIで遊べる、1人用ローグライクボードゲームです。  
ランダム生成されるフロアを進み、敵を倒して成長しながら、より深い階層を目指します。

## 実行方法

```bash
python game.py
```

## タイトル画面

起動後はタイトル画面が表示され、次の項目を選べます。

- **Start**: ゲーム開始
- **Change Difficulty**: 難易度変更
  - Easy / Normal / Hard / Lunatic
- **Quit**: 終了

## 操作方法

- `w` / `a` / `s` / `d`: 移動（敵にぶつかると攻撃）
- `.`: 待機
- `t`: 魔法（Arcana）を使用
- `i`: インベントリ表示
- `u`: アイテム使用
- `k`: スキルツリーを開く（SP使用）
- `h`: ヘルプ表示

## 画面表示とアイコン

- `#`: 壁
- `.`: 床
- `,`: ボスレーザー予告ライン
- `@`: プレイヤー
- `E`: 敵
- `M`: 中ボス
- `B`: ボス
- `I`: アイテム
- `F`: 友好キャラクター
- `>`: 階段

ステータスは3行で表示されます。

1. Floor / Moves
2. HP / MP / SP
3. Lv / XP / ATK / DEF

## 基本ルール

- 初期ステータス: HP 10 / MP 5 / ATK 3 / DEF 1
- ダメージ計算: `max(1, attacker_atk - defender_def)`
- HPが0以下でゲームオーバー
- スコアは到達階層・レベル・最終ステータス・所持Arcanaのレアリティで計算

### 階層と進行

- 階段 `>` で次のフロアへ進行
- フロア5, 15, 25... は中ボス階（中ボス撃破まで階段封印）
- フロア10, 20, 30... はボス階（ボス撃破まで階段封印）
- 11, 21, 31... で新しい周回に入り、次が強化されます
  - 敵ステータスの周回補正
  - マップ横幅の拡張

## 成長要素

### レベルアップ

- 敵撃破やフロア全敵撃破ボーナスでXP獲得
- レベルアップ時:
  - 最大HP/最大MP/ATK/DEFが成長
  - HP/MP全回復
  - SPを1獲得

### スキルツリー（`k`）

- `v` Vitality: 最大HP +3
- `s` Strength: ATK +2
- `g` Guard: DEF +2
- `a` Arcane: ランダムArcanaを習得（または入れ替え）

## アイテム

### 種類

- `Potion`: HP回復
- `Power`: ATK上昇
- `Shield`: DEF上昇
- `Ether`: MP回復
- `Throwing axe`: 1方向直線の敵に割合ダメージ
- `Bomb`: 全敵に割合ダメージ

### レアリティ

- Common / Uncommon / Rare / Epic / Legendary
- 高レアほど出現率は低く、効果は高い

## Arcana（魔法）

Arcanaは**1つだけ所持**できます（新規習得時は入れ替え確認あり）。

- `Comet Missile`: 直線上の敵1体を攻撃
- `Flare Curtain`: 周囲1マスの敵を同時攻撃
- `God's Wrath`: 1ターン詠唱後、高威力攻撃
- `Healing`: 自身のHP回復
- `Vampire Kiss`: 隣接敵を攻撃して吸収回復
- `Frugal soul`: 次に使うアイテムの消費を確率で無効化

## 友好キャラクター（`F`）

各フロアに以下3人が出現します（それぞれ1回まで）。

- **merchant**
  - 任意のアイテム1つを、ランダムな別アイテムに交換
- **technician**
  - 任意のアイテム1つを、ランダムなArcanaに変換
- **maze traveler**
  - 次に階段を使った際の移動先を、現在階層の
    - 2倍の階層
    - 半分の階層
    のどちらかに変更

## ボス行動

ボスは以下の行動を取ります。

- 近接攻撃
- エリート敵の召喚
- レーザー予告→次ターン照射
- HP低下時の一度だけの強化（回復＋能力上昇）
